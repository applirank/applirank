# System Requirements

Reqcore is designed to be lightweight. A small VPS can comfortably handle most hiring workflows.

## Development

| Requirement | Minimum |
|-------------|---------|
| **Node.js** | 18.20+ (LTS 20+ recommended) |
| **Docker** | 20.10+ with Docker Compose v2 |
| **RAM** | 4 GB |
| **Disk** | 2 GB free |
| **OS** | macOS, Linux, or Windows (via WSL2 / Git Bash) |

## Production (Self-Hosted)

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| **CPU** | 1 vCPU | 2 vCPU |
| **RAM** | 2 GB | 4 GB |
| **Disk** | 10 GB | 20 GB+ (depends on resume storage) |
| **OS** | Ubuntu 22.04+ / Debian 12+ | Ubuntu 24.04 LTS |
| **Docker** | 20.10+ | Latest stable |
| **PostgreSQL** | 15+ | 16 |

## Networking

| Port | Service | Required |
|------|---------|----------|
| 80 | HTTP (redirect to HTTPS) | Yes |
| 443 | HTTPS (reverse proxy) | Yes |
| 3000 | Nuxt app (internal) | Internal only |
| 5432 | PostgreSQL (internal) | Internal only |
| 9000 | MinIO S3 API (internal) | Internal only |

## Browser Support

Reqcore supports current versions of modern browsers:

- Chrome / Chromium 90+
- Firefox 90+
- Safari 15+
- Edge 90+

## Infrastructure Providers

Reqcore has been tested on:

| Provider | Reference Setup | Monthly Cost |
|----------|----------------|-------------|
| **Hetzner Cloud** | CX23 (2 vCPU, 4 GB RAM) | ~€5/mo |
| **DigitalOcean** | Basic Droplet (2 vCPU, 4 GB) | ~$24/mo |
| **AWS EC2** | t3.medium (2 vCPU, 4 GB) | ~$30/mo |
| **Railway** | Managed (auto-scale) | Usage-based |

Any Linux VPS with Docker support will work.
