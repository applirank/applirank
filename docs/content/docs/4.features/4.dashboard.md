# Dashboard

The recruiter dashboard provides an at-a-glance overview of your organization's hiring activity.

## Dashboard Components

### Stat Cards

Summary statistics displayed at the top of the dashboard:

- **Total Jobs** — Count of all jobs in the organization
- **Open Jobs** — Count of currently open positions
- **Total Candidates** — Count of all candidates in the organization
- **Total Applications** — Count of all applications across jobs

### Pipeline Breakdown

Visual breakdown of applications by pipeline stage:

- New
- Screening
- Interview
- Offer
- Hired
- Rejected

### Recent Applications

A feed of the most recent applications, showing:

- Candidate name
- Job title
- Application date
- Current stage

### Top Active Jobs

The jobs with the most activity (recent applications), with quick links to the pipeline view.

## Navigation

The dashboard uses a sidebar layout with:

- **Organization switcher** — Switch between organizations
- **Dashboard** — Overview (current page)
- **Jobs** — Job management
- Dynamic job context tabs (when viewing a specific job):
  - Overview
  - Pipeline (Kanban)
  - Candidates (table)
  - Application Form (custom questions)

## Layouts

Reqcore uses three layouts:

| Layout | Used For |
|--------|----------|
| `dashboard` | All authenticated dashboard pages (sidebar + full-width main) |
| `auth` | Sign-in and sign-up pages (centered card) |
| `public` | Public pages like the job board |

The landing page, roadmap, and blog use a no-layout dark theme.
