# Internationalization

Reqcore uses `@nuxtjs/i18n` for internationalization with [Crowdin](https://crowdin.com) for community translation management.

## Supported Languages

| Language | Code | Status |
|----------|------|--------|
| English | `en` | Complete (source) |
| Norwegian Bokmål | `nb` | Complete |
| Spanish | `es` | Partial |
| French | `fr` | Partial |
| German | `de` | Partial |
| Vietnamese | `vi` | Partial |

## How It Works

### URL Strategy

Reqcore uses the `prefix_except_default` strategy:

| URL | Language |
|-----|----------|
| `/jobs` | English (default) |
| `/nb/jobs` | Norwegian |
| `/es/jobs` | Spanish |

### Translation Files

Translation strings are stored as JSON files in the `i18n/locales/` directory:

```
i18n/
└── locales/
    ├── en.json    # English (source of truth)
    ├── nb.json    # Norwegian Bokmål
    ├── es.json    # Spanish
    ├── fr.json    # French
    ├── de.json    # German
    └── vi.json    # Vietnamese
```

### Using Translations

In Vue templates:

```vue
<template>
  <h1>{{ $t('dashboard.title') }}</h1>
  <p>{{ $t('jobs.count', { count: jobs.length }) }}</p>
</template>
```

In `<script setup>`:

```ts
const { t } = useI18n()
const title = t('dashboard.title')
```

## Crowdin Integration

Translations are managed via Crowdin with a native GitHub integration:

### Translation Workflow

1. Developer adds English strings to `i18n/locales/en.json`
2. Crowdin detects the change and syncs the source file
3. Translators work on translations in Crowdin
4. Crowdin creates a PR with updated translation files

### Manual Sync Commands

```bash
# Upload source strings to Crowdin
npm run i18n:crowdin:upload

# Download latest translations
npm run i18n:crowdin:download

# Both upload and download
npm run i18n:crowdin:sync
```

## Contributing Translations

Want to help translate Reqcore? Join the translation effort on Crowdin:

1. Visit the [Reqcore Crowdin project](https://crowdin.com/project/reqcore)
2. Choose a language
3. Start translating

No development experience is needed — Crowdin provides a visual editor.
